# 🛒 C++ Shopping Cart Console Application

This is a simple command-line based shopping cart system written in C++. It simulates an e-commerce experience with user signup/login, product browsing by category, cart management, and order placement.

---

## 📌 Features

- 🔐 **User Authentication**: Basic sign-up and sign-in system.
- 🛍️ **Product Catalog**: Products are organized into categories like Clothes, Jewelry, Makeup, and Shoes.
- 🛒 **Shopping Cart**: Add, view, and clear items in your shopping cart.
- 📦 **Order Management**: Place orders and review order history.
- 🧾 **Clean CLI Interface**: Simple text-based menu navigation for ease of use.

---

## 🧠 How It Works

### 👤 User Flow:
1. Sign up or sign in with a username.
2. Browse products by category.
3. Add products to the cart using their product ID.
4. View cart contents and total.
5. Place an order to finalize your cart.
6. View your order history.
